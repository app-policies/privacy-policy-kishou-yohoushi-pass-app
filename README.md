# 気象予報士Pass プライバシーポリシー

最終更新日: 2026年7月3日

気象予報士Pass（以下「本アプリ」）は、気象予報士試験の学科試験対策を支援する非公式の学習アプリです。本アプリは、学習機能、広告表示、アプリ内課金、品質改善のために、端末上の学習履歴および外部サービスが提供する識別子等を利用する場合があります。

本アプリは、ログインやアカウント作成を必要としません。

## 収集する情報

- 学習履歴、回答履歴、正誤履歴、苦手問題、連続学習日数など、アプリ内で保存される学習データ
- 文字サイズなどのアプリ設定
- アプリの起動、学習開始、回答、学習完了、実戦演習開始・完了、購入画面表示などの利用状況イベント
- 広告配信や広告効果測定に必要な端末情報、広告表示に関する情報
- アプリ内課金の購入状態を管理するために必要な情報
- Apple が提供するクラッシュログ、診断情報などの技術情報

本アプリは、氏名、住所、電話番号、メールアドレス、連絡先、正確な位置情報を、アカウント情報として独自に取得しません。

## 利用目的

- 問題演習、復習、学習進捗表示の提供
- 誤答優先学習、苦手問題、連続学習日数などの学習機能の提供
- 広告表示
- アプリ内課金の購入状態の確認、復元、管理
- 不具合調査、品質改善、利用状況の把握

## 学習データの保存

学習履歴、回答履歴、苦手問題、連続学習日数などの学習データは、原則として利用者の端末内に保存されます。

本アプリは、学習履歴を管理するための独自アカウントサーバーを運営していません。端末の削除、アプリの削除、端末変更などにより、端末内の学習データが失われる場合があります。

## Firebase Analytics について

本アプリでは、利用状況の把握と品質改善のために Firebase Analytics を利用します。

Firebase Analytics では、アプリ起動、学習開始、回答、学習完了、実戦演習開始・完了、購入画面表示などのイベントを送信する場合があります。

本アプリは、Firebase Analytics に問題本文、選択肢本文、解説本文、自由入力、氏名、メールアドレス、個別相談内容、生の取引ID、広告・課金・Firebaseの内部識別子を送信しません。

初期リリースでは、Firebase Auth、Firestore、Realtime Database、Cloud Functions、Storage、Remote Config、Crashlytics、Cloud Messaging、App Check は使用しません。

## 広告について

本アプリでは、Google Mobile Ads を利用して、バナー広告およびインタースティシャル広告を表示する場合があります。

Google Mobile Ads では、広告配信、広告効果測定、不正防止などのために、端末情報、広告表示に関する情報、利用状況に関する情報が利用される場合があります。これらの情報の取り扱いは、Google のポリシーに従って管理されます。

初期リリースでは、リワード広告による問題追加解放は提供しません。

## アプリ内課金について

本アプリでは、非消耗型のアプリ内課金により、全問題の利用、広告非表示、詳しい解説、選択肢ごとの解説、苦手問題、実戦演習などの機能を提供する場合があります。

購入処理は Apple の仕組みを通じて行われ、購入状態の管理には RevenueCat を利用します。RevenueCat では、購入状態の確認、復元、課金機能の提供に必要な識別子や購入情報が取り扱われる場合があります。

## 第三者サービス

本アプリでは、広告配信、課金管理、品質改善のために、以下の第三者サービスを利用する場合があります。各サービスで取り扱われる情報は、それぞれのプライバシーポリシーに従って管理されます。

- Firebase Analytics
- Google Mobile Ads
- RevenueCat
- Apple が提供する App Store、アプリ内課金、クラッシュログ、診断関連サービス

## 外部送信

広告配信、課金管理、利用状況分析、クラッシュ解析、品質改善のため、第三者サービスに端末情報、広告表示に関する情報、購入情報、クラッシュ情報、診断情報などが送信される場合があります。

本アプリは、利用者が入力した氏名、住所、電話番号、メールアドレス、連絡先、正確な位置情報を独自に収集して外部送信する機能を持ちません。

## 広告識別子とトラッキング

本アプリは、初期リリースでは IDFA を使用しません。App Tracking Transparency の許可要求も行いません。

広告は非パーソナライズ広告を前提とし、他社のアプリやWebサイトを横断して利用者を追跡する目的では利用しません。

## 公式機関との関係

本アプリは、一般財団法人気象業務支援センター、気象庁、国土交通省、その他の公的機関または試験実施団体が提供・公認・監修するアプリではありません。

本アプリの問題および解説は、公開されている試験概要、試験範囲、学科試験の出題分野を参考に、スマホ学習向けに独自作成したものです。公式問題、過去問題、解答例、公式PDF本文、公式天気図、解析図、公式図表、公式ロゴをそのまま収録しているものではありません。

本アプリは、実技試験の答案添削、天気図読解、個別地点の天気予報、防災判断、警報・注意報判断、業務予報支援、気象コンサルティングを提供しません。
